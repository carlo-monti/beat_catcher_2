Detailed Description

MODE_SWITCH module handles the changing of the main mode of Beat Catcher. There are four main modes of operation: MODE_TAP MODE_PLAY MODE_SETTINGS MODE_SLEEP The mode module has a main task that is exposed and can be notified using a value from mode_switch_task_notify_value. In this way you can ask the task to change the mode.
