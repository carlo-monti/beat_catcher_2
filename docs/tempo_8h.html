Detailed Description

Tempo module evaluates the need of a change in the bpm of the midi clock. For every onset, the sync module calculates the IOI with the preceding onsets of the two bars and calculates the accuracy. If the highest accuracy found is higher than the threshold the module sends a message to the clock module asking to set delta_tau_tempo value. The module starts its job when notified by the Sync module.
