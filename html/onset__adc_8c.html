◆ led_blink_init()

void led_blink_init ( )

Init function for the blinking led action.

When an onset is triggered, the function blink_led is called. The led is than turned on and a timer is started to trigger an interrupt that will turn it off after a duration period. This function inits the GPIO and the timer.
